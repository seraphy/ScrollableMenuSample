Java SwingのJMenuを拡張して、スクローラブルなメニューにするデモンストレーションです。

基本原理は、okwaveの以下のリンク先にあるようにJMenuにJMenuItemを沢山登録したあとで、
表示させたい範囲だけVisibleにして、それ以外をInvisibleにし、スクローラーのup/downアイテムを
選択することで表示範囲を上下させる、というものです。

オリジナルとは以下の点で異なります。

・ 矢印はPNG画像アイコン
・ 矢印をクリックすることで上下にスクロールする。
・ 矢印を押しっぱなしで自動スクロール、数回スクロール後に高速スクロールする
・ メニューを開いたときにウィンドウサイズにあわせて表示可能なアイテム数を動的に変更する。
